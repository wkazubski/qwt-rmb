
The Qwt-rmb Library
----------------------

      Qwt-rmb is an extension to the libraries of the Qwt Project.  

      The Qwt-rmb library contains modified state machine that supports
      rigt mouse clicks in Qwt.

Installation
------------

      Read INSTALL how to build and install Qwt.
